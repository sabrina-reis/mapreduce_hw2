# Map Reduce HW 2
This code is for HW2 in CSE 239: Cloud Computing at UCSC, which uses Docker to implement a simple MapReduce approach.

## Steps to Run
```bash
git clone https://github.com/sabrina-reis/mapreduce_hw2.git
cd mapreduce_hw2
NUM_WORKERS=4 docker compose up --scale worker=4
```
